Ayuda - Algoritmo Johnson

Guía de Uso

Esta página te permite visualizar el algoritmo Johnson y realizar diversas operaciones. Aquí tienes una guía para comenzar:

  1. Creación de nodos: Haz clic en la pizarra para crear nodos. Cada nodo representa un punto en tu gráfico.
  2. Asignación de valores: Después de crear un nodo, puedes hacer doble clic en él para asignarle un valor o etiqueta.
  3. Conexiones: Para crear conexiones entre nodos, arrastra líneas desde un nodo a otro.
  4. Operaciones: Puedes realizar cálculos y operaciones específicas según tu necesidad, como encontrar caminos o realizar análisis.
  5. Modificación y eliminación: Para modificar nodos o conexiones, haz clic derecho sobre ellos. También puedes eliminarlos.

¡Explora y disfruta de la visualización del algoritmo Johnson!
